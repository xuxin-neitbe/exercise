```bash

DASHSCOPE_API_KEY=sk-55df090cff7449758924b6390e3df600
DASHSCOPE_MODEL=tongyi-xiaomi-analysis-pro
```

### 2. 修改 API Key

如需修改 API Key，只需编辑 `.env` 文件：

```bash
# 打开 .env 文件
# 修改 DASHSCOPE_API_KEY 为你的 API Key
```

**注意：** 代码会自动从 `.env` 文件加载环境变量，无需修改代码！

## 🔧 代码说明

### 环境变量加载

所有代码都使用 `python-dotenv` 自动加载 `.env` 文件：

```python
# agent.py
try:
    from dotenv import load_dotenv
    load_dotenv()
except ImportError:
    pass  # 如果没有安装 python-dotenv，使用系统环境变量
```

### 读取配置

```python
# 从环境变量读取，无需硬编码
api_key = os.getenv("DASHSCOPE_API_KEY")
model_id = os.getenv("DASHSCOPE_MODEL", "tongyi-xiaomi-analysis-pro")
base_url = os.getenv("DASHSCOPE_BASE_URL", "https://dashscope.aliyuncs.com/compatible-mode/v1")
```

## ⚠️ 注意事项

1. **`.env` 文件不要提交到 Git**（应添加到 `.gitignore`）
