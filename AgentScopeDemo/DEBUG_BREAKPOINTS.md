## 📋 断点设置方法

### 方法 1：鼠标点击
1. 打开 `main_cn.py` 文件
2. 将鼠标移动到行号左侧
3. 点击会出现红色圆点（断点）
4. 再次点击可删除断点

### 方法 2：快捷键
1. 将光标移动到目标行
2. 按 `F9` 键切换断点

### 方法 3：命令面板
1. 将光标移动到目标行
2. 按 `Ctrl+Shift+P`
3. 输入 "Debug: Toggle Breakpoint"
4. 回车确认

---

## 🎮 调试步骤

1. **设置断点** - 在以上位置添加断点
2. **选择配置** - 选择 "Python Debugger: 三国狼人杀"
3. **启动调试** - 按 `F5`
4. **观察执行** - 程序会在断点处暂停
5. **查看变量** - 在变量面板查看当前值
6. **单步执行** - 使用 F10/F11 逐步执行
7. **继续运行** - 按 F5 继续到下一个断点

## 🔍 调试技巧

### 查看变量
- 鼠标悬停在变量上查看值
- 在"运行和调试"面板查看局部变量
- 使用"监视"面板添加自定义表达式

### 条件断点
1. 右键点击断点
2. 输入条件表达式
3. 只有条件满足时才会暂停

### 日志断点
1. 右键点击断点
2. 选择"记录消息"
3. 输入日志消息
4. 断点会记录日志但不停止
