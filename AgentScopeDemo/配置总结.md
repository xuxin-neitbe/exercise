## 📋 问题解答

### 1️⃣ PyPI 源是什么？为什么每个项目都需要？

**PyPI 源** = Python 包的下载服务器地址

- **官方源**: `https://pypi.org` (Python Package Index)
- **镜像源**: 清华、阿里、腾讯等提供的国内镜像

**为什么需要配置？**
- 官方源在国外 → 国内访问慢
- 镜像源在国内 → 速度快，但包可能不全
- `agentscope` 等较新的包在镜像源可能没有

---

### 2️⃣ 为什么默认总是去清华源找？

**原因**: 你的 pip 全局配置文件中设置了清华源

**之前的配置**:
```ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn
```

这导致 pip **只**从清华源下载包，找不到 agentscope。

---

### 3️⃣ 能否一劳永逸解决这个问题？

**✅ 已解决！**

现在的配置策略：
```
1. 优先从官方源 (pypi.org) 下载
   ↓ (如果失败或超时)
2. 自动尝试清华源 (pypi.tuna.tsinghua.edu.cn)
   ↓
3. 安装成功 ✅
```

**优势**:
- ✅ 包最全（官方源什么都有）
- ✅ 有备用（官方源失败自动切清华源）
- ✅ 全局配置（所有项目自动生效）
- ✅ 无需每个项目单独配置
