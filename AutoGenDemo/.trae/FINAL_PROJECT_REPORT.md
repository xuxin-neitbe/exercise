## 🚀 使用指南

### 1. 使用 Skills

**代码模板 Skill：**
```
用户：创建一个新项目
→ 自动触发 code-template Skill
→ 复制模板并替换变量
```

**配置生成 Skill：**
```
用户：生成 DevOps 配置
→ 自动触发 config-generator Skill
→ 生成云效流水线、K8s 配置
```

### 2. 使用 Agents

**Project Builder Agent：**
```
用户：我要创建一个新项目
→ 6 步交互式问答
→ 自动创建完整项目
```

**ACR Registry Agent：**
```
用户：配置 ACR 私有仓库
→ 自动推送基础镜像
→ 修改 Dockerfile
→ 生成云效配置
```

### 3. 使用模板

**直接使用模板：**
```bash
# 运行测试脚本
node examples/test-template-copy.js

# 示例项目位置
cd examples/test-shop

# 安装依赖
npm install

# 启动开发
npm run dev
```
